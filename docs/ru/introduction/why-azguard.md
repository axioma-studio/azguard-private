# Что такое AzGuard?

AzGuard — это **code-first RBAC-пакет для Laravel**. В отличие от библиотек, где права хранятся в базе данных в виде строк, AzGuard выражает права как **PHP enum-кейсы**, а роли как **PHP-классы**. База данных хранит только связки `user→role` — каталог прав живёт в коде.

## Ключевые принципы

**Права в коде, не в БД** — каталог прав версионируется в Git, проходит code review, поддаётся статическому анализу.

**Роли как классы** — каждая роль — это отдельный PHP-класс со списком enum-кейсов. IDE понимает структуру.

**Нативная интеграция с Gate** — `@can`, `Gate::allows()`, `$this->authorize()` работают без изменений.

## Когда использовать AzGuard

- Команды, которые хотят code review для изменений прав
- Проекты с несколькими панелями (admin + app + api)
- Приложения, где нужна статическая типизация прав
- Среды Octane / Kubernetes с требованием к stateless-архитектуре
