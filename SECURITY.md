# Security Policy

## Supported Versions

AzGuard is pre-1.0. Security fixes land on the latest `0.x` release and `main`.

## Reporting a Vulnerability

Please **do not** open a public issue for security problems.

Report vulnerabilities privately via [GitHub Security Advisories](https://github.com/axioma-studio/azguard/security/advisories/new),
or by email to **dv.vostrikov@gmail.com**.

Include where possible:

- a description of the issue and its impact,
- affected package(s) and version(s),
- reproduction steps or a proof of concept.

You can expect an initial acknowledgement within a few business days. Once a fix
is ready we will coordinate a release and credit you in the advisory unless you
prefer to remain anonymous.
