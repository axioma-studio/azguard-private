# Шаблон команды /execute — скопировать в .claude/commands/execute.md проекта

```markdown
---
description: Выполнение готового плана из файла (вторая сессия паттерна Plan→Clear→Execute)
model: sonnet
argument-hint: <путь к плану из .claude/plans/>
---

Выполни план: $ARGUMENTS

1. Прочитай файл плана. Это единственный источник контекста задачи —
   не пытайся восстановить историю обсуждения.
2. Выполняй шаги по порядку; читай только файлы, названные в плане
   (плюс то, что они явно требуют).
3. Если план противоречит реальному коду — остановись и опиши расхождение,
   не импровизируй.
4. Заверши верификацией из плана и компактным финальным отчётом:
   сделано / не сделано и почему / изменённые файлы / команды проверки.
```
